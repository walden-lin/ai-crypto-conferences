## Responsive Breakpoints

### Desktop (1200px+)
- 3-column event grid
- Full navigation with all features
- Sidebar filter panel
- Expanded event cards with full details

### Tablet (768px - 1024px)
- 2-column event grid
- Collapsible navigation
- Bottom sheet filter panel
- Compact event cards

### Mobile (320px - 768px)
- 1-column event grid
- Hamburger menu navigation
- Full-screen filter modal
- Stacked event card layout

## Interactive Elements

### Hover States
- Event cards lift slightly (2px)
- Shadow increases
- Border color changes to accent
- Smooth transitions (0.2s ease)

### Focus States
- Clear outline for keyboard navigation
- High contrast colors
- Accessible focus indicators

### Loading States
- Skeleton screens for event cards
- Shimmer effects for images
- Progressive loading of content

---

*These wireframes provide a clear visual guide for implementing the Conference Summary Website across all device sizes. Focus on clean, minimalist design with smooth interactions and excellent accessibility.*
